Presentations

  • Code GPU with CUDA

    This is a list of presentations for course about NVIDIA GPU architecture. The main aim of the course is to explain the details and unique properties of the architecture and to give advice about approaches to optimization.

    The first presentation explains key properties of the architecture. The second – describes the ideas of programming model which is named "Cooperative Thread Arrays". The third presentation highlights the main optimization principle. The fourth presentation describes the memory architecture in the very details. The fifth – gives concrete advice for optimizers how to deal with SIMT GPUs. The sixth presentation gives more concrete examples for optimizations. The last one shows helpful approaches to identify performance limiters.

    1. SIMT - NVIDIA GPU architecture
    2. CTA programming model
    3. Device code optimization principle
    4. Memory subsystem
    5. Optimizing memory & control flow
    6. Applying optimization techniques
    7. Identifying performance limiters
